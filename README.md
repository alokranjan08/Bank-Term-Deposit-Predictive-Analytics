# Bank Term Deposit Predictive Analytics

This repository contains a predictive analytics project focused on bank term deposits.

## Project Structure

- **`finalca_upgraded.ipynb`**: The main Jupyter Notebook containing the complete analysis, data preprocessing, and machine learning models.
- **`finalca.ipynb`**: An older/alternative version of the analysis.
- **`BankAdditional.csv`**: The dataset used for training and evaluating the models.

## Dataset Information

The data used in this project is the **Bank Marketing Dataset**, which is publicly available from the UCI Machine Learning Repository:
[Bank Marketing Dataset (UCI)](https://archive.ics.uci.edu/dataset/222/bank+marketing)
